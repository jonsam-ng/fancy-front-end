## 如何更高效的搜索 React 源码仓库？

在 vscode 搜索栏添加如下选项：

```txt
files to include: src/react/packages/
files to exclude: *test*.js,*.coffee,*test.ts,*.d.ts,*.md,*.snap,*.old.js
```

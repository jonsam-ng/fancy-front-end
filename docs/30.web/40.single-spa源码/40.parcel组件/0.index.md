一个single-spa 的 parcel，指的是一个与框架无关的组件，由一系列功能构成，可以被手动挂载，而无需关心由哪种框架实现。注册 Parcels 和注册应用的api一致，不同之处在于 parcel 组件需要手动挂载，而不是通过路由匹配被激活。
